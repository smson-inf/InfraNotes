
# CPU
## CPU 구성
### ALU (Arithmatic Logic Unit) 
산술논리장치
### Control Unit
제어장치. 주변 기기 (그래픽 카드, 메모리 등) 제어

### Registers
레지스터. 기억 장치 (Memory Unit)
메인보드에 장착된 메모리(RAM)과 정보를 주고 받는다. 이 경로가 넓을수록 ( = 비트가 높을수록 ) 한 번에 처리할 수 있는 데이터 양이 많아져 처리 성능이 높아진다.

## Bit 란
- CPU 구성요소 간 데이터 교환의 단위
- CPU, OS, App이 같은 비트로 구성되는 것이 중요


> Intel은 서버용 CPU로 Xeon(제온) > Skylake 로 넘어가는 추세


> AMD는 서버용 CPU로 EPYC 시리즈를 사용하며, 가정용으로 Ryzen (라이젠)을 사용한다.
